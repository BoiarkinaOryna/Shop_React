
## Сторінка «Мої замовлення»

Сторінка відображає список замовлень користувача.
Містить номер замовлення, дату, статус, список товарів та загальну суму.
Передбачено перегляд деталей кожного замовлення.

## Сторінка «Адреси доставки»

Сторінка для керування адресами доставки користувача.
Можна додавати нові адреси, редагувати існуючі та вибирати основну адресу.

## Успішне оформлення замовлення
Сторінка підтвердження замовлення. Показує повідомлення про успішне оформлення, номер замовлення, інформацію про подальшу обробку та кнопку переходу на головну сторінку.

## Редагування товарів 
Модальне вікно керування кошиком. Дозволяє змінювати кількість товарів, видаляти позиції, переглядати знижки, загальну суму та зберігати або скасовувати зміни.

## Оформлення замовлення
Сторінка checkout з формою контактних даних, вибором доставки та оплати. Відображає список товарів, знижки, вартість доставки та фінальну суму до оплати.

## Сторінка товару 
Сторінка дрона з акцентом на камеру, час польоту та памʼять. Містить візуальні матеріали, технічні характеристики, опис переваг і блок схожих товарів.

## Сторінка «Авторизація»
Форма для входу в особистий кабінет. Містить поля для введення Email та пароля. Користувач може скористатися функцією відновлення пароля або перейти до створення нового акаунту.

## Сторінка «Реєстрація»
Призначена для створення нового профілю користувача. Потрібно вказати ім'я, електронну пошту та двічі ввести пароль для підтвердження. Містить посилання на умови публічного договору.

## Сторінка «Відновлення пароля»
Використовується у випадку, якщо користувач забув дані для входу. Потрібно ввести лише Email, на який система надішле інструкції для скидання поточного пароля.

## Сторінка «Новий пароль»
З’являється після переходу за посиланням з пошти. Дозволяє безпечно встановити новий пароль, ввівши його двічі у відповідні поля з можливістю перевірки введених значень.

## Модальні вікна статусів
Інформаційні вікна, що підтверджують успішність дій. Повідомляють про успішне створення акаунту або зміну пароля, надаючи кнопку для швидкого переходу до роботи з сайтом.

## Сторінка «404 Not Found»
Сторінка, яка відображається, коли користувач намагається перейти за неіснуючим або видаленим посиланням. Вона інформує про помилку та зазвичай містить кнопку повернення на головну сторінку для продовження роботи з сайтом.

## Головна сторінка
Центральна сторінка сайту, що знайомить користувача з брендом. Вона містить банер із ключовою перевагою, розділ «Про нас», блок «Нове на сайті» та прев'ю каталогу товарів. Також на сторінці відображена статистика успіху компанії та швидкі посилання у футері.

## Сторінка «Контакти»
Містить усю необхідну інформацію для зв'язку: номер телефону, електронну пошту, фізичну адресу та графік роботи. Для зручності користувачів інтегрована форма зворотного зв'язку для відправки повідомлень безпосередньо з сайту та посилання на соціальні мережі.
